# Spring Boot的主要优点

  • 为所有Spring开发者更快的入门
  • 开箱即用，提供各种默认配置来简化项目配置
  • 内嵌式容器简化Web项目
  • 没有冗余代码生成和XML配置的要求

其实就是提供更快速的开发，减少各种开发jar的配置，xml等繁杂的配置，使开发者专注业务开发
